# CoStudy - AI-Driven Collaborative Learning Platform

## Vision
An intelligent learning platform that combines AI tutoring with collaborative study features, designed for Indian students preparing for competitive exams.

## Core Features

### 1. AI Tutor
- Personalized AI conversations for doubt clearing
- Subject-specific tutoring (Math, Science, Languages, etc.)
- Adaptive learning based on student performance

### 2. Collaborative Study
- Virtual study rooms
- Peer matching based on subjects/goals
- Real-time collaboration tools

### 3. Exam Preparation
- Mock tests with AI-generated questions
- Performance analytics
- Personalized study plans

### 4. Progress Tracking
- Dashboard with learning metrics
- Goal setting and tracking
- Achievement system

## Tech Stack
- **Frontend:** React + TypeScript + TailwindCSS
- **Backend:** Node.js + Supabase
- **AI:** Ollama (local) / OpenAI fallback
- **Auth:** Supabase Auth

## Target Users
- Indian students (Class 6-12 + competitive exams)
- NEET, JEE, UPSC aspirants
- Language learners

## Design
- Premium, clean interface
- FETS Yellow accent color
- Mobile-first responsive design
